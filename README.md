# ПК «Умный Excel» 📊

Программный комплекс для автоматической проверки, нечёткого сопоставления данных и консолидации отчётных форм.

## 🚀 Технологический стек
* Язык: Python 3.10
* Интерфейс: CustomTkinter (Современный GUI с поддержкой тем)
* Алгоритм сопоставления: Расстояние Левенштейна (RapidFuzz на C++)
* База данных: SQLite3 (логирование сессий и аудит действий)

## 🛠 Быстрый запуск в один клик (Windows)

Вам не нужно вручную настраивать окружение и устанавливать библиотеки. Всё автоматизировано.

1. Клонируйте репозиторий:
`bash
git clone [https://github.com/Sofia-claire/KURSACH.git](https://github.com/Sofia-claire/KURSACH.git)
cd KURSACH

 3. **Запустите проект:**
   Просто дважды кликните по файлу **start.bat** в корневой папке. Скрипт сам проверит наличие Python, автоматически установит все недостающие зависимости (customtkinter, rapidfuzz, pandas, openpyxl) и запустит графический интерфейс.
